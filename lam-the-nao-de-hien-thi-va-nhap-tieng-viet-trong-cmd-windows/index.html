Làm thế nào để hiển thị và nhập tiếng Việt trong cmd Windows

Bước 1: Mở cmd gõ chcp 65001 https://msdn.microsoft.com/en-us/library/windows/desktop/dd317756%28v=vs.85%29.aspx

Bước 2: Chuột phải vào thanh phía trên cmd (thanh này gọi là thanh gì nhể? :smile: ) , chọn properties

Bước 3: Chọn font Concolas

Đến đây thì cmd đã có thể hiển thị chữ có dấu (UTF-8)

Bước 4: Tải tool hỗ trợ (có nhiều cách để nhập kí tự unicode nhưng xem ra cách này nhanh nhất ) Tải tool này về: http://www.fileformat.info/tool/unicodeinput/index.htm Bật nó lên, khi muốn gõ chữ nào thì ấn Alt + + thì tool hiện ra, gõ code của chữ mình cần gõ vào -> Enter

Cách đơn giản nhất là viết ra notepad, xong copy vào :smile:

Nhờ kĩ thuật này nên DNHconsole 2.0 sẽ hỗ trợ hiển thị tiếng Việt :v: [1]: http://www.fileformat.info/tip/microsoft/enter_unicode.htm [2]: http://www.utf8-chartable.de/unicode-utf8-table.pl
